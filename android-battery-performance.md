# Raising the Bar on Battery Performance  

Battery performance is critical for user experience in Android devices. Here are best practices for optimizing battery usage:

## Best Practices for Battery Optimization  
1. **Use Doze and App Standby**: These features help reduce battery usage when a device is idle.
2. **Optimize Background Services**: Limit background activity to conserve battery life.
3. **Use JobScheduler**: Schedule tasks efficiently to minimize wake-up calls.
4. **Reduce Location Updates**: Use location services judiciously to save power.
5. **Measure Battery Use**: Use tools to measure how much battery apps use to identify optimization opportunities.  

For more details, visit the [Android Developer Blog](https://android-developers.googleblog.com/2025/11/raising-bar-on-battery-performance.html).
